# 🎆 Animate Fireworks with JavaScript & Canvas

An interactive fireworks animation built using HTML5 Canvas and JavaScript.

This project features:
- 🚀 Rocket launch animation before explosion
- 💥 Realistic explosion physics
- ✨ Glowing spark particles with trail effects
- 🌈 Colorful dynamic bursts
- 📱 Responsive full-screen canvas
- 🎉 Smooth animation using requestAnimationFrame

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Canvas API
